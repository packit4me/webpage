I have a problem.

I hate it when I want to install software and its not packaged using my distributions default package manager.

Typically I'm on RPM based distributions, but I've run across the same thing elsewhere. There are plenty of repositories available that have lots of software, but unfortunately there are some bits that don't make it into these.

  1. First is commercial software. I can't redistribute most of that for legal reasons, but I can help facilitate finding good spec files for building your own package! If you package a commercial software and have a spec to share, contact me!
  2. The other is Puppet modules. What? package puppet modules? Why would someone do that? Well.. its cause if you like managing your systems with RPMs there almost no clean ways for you to manage the Puppet modules you download from Pupept Forge. There is a new system called 'Pulp' that does allow you to manage your own private Puppet Forge, and thats still a pretty good idea. However, I wanted to give this a shot sooo..
Introducing the PackIt4.Me Puppet Module Yum repository!

How to use it? Well when its ready you will just install this rpm. You can then download any puppet module, and its dependencies, with yum!
